# Python Key Logger

DISCLAIMER:
This project is purely for educational purposes. I created it to learn how keyboard activities and clipboards can be monitored. This is not intended to monitor actual user activities.

</div>

---

## Description

A simple key logger project that will detect keyboard activities as well as contents saved on the user's clipboard. It will save these onto two seperate txt files.


---

## Features

<ul>
  <li>Monitors keyboard activities</li>
  <li>Monitors clipboard activities</li>
  <li>Saves activities onto txt files</li>
</ul>
